Introduction

This game is for:
  • Selective Attention: Focus on a particular object while avoiding distractions
  • Visual Recognition: Perceiving an object's physical properties and identifying it

This application is Java8 based. UI is designed in Swing.
No third party library is used.
MatchPair is a game which is freely available. It offers:

  1. User can select to practice times tables
  2. User can select table range between 1 and 30 to practice
  3. Game will run for 1 minute and checks for correct/wrong/not-answered replies
  4. User can set how many question he wants to practice in a minute
  5. Latest 30 scores will be visible in history
  6. User can Start/Stop game any time; On Stopping game will not be counted
  7. User can view details of a game by double-clicking on history row in a colorful way
