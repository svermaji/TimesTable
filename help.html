Introduction

This game is for:
  • Selective Attention: Focus on a particular object while avoiding distractions
  • Visual Recognition: Perceiving an object's physical properties and identifying it

This application is Java8 based. UI is designed in Swing.
No third party library is used.
MatchPair is a game which is freely available. It offers:

  1. There are multiple levels in this game
  2. Shows graph with tooltip and options to change it
  3. Each user will have its own password
  4. User can be changed from existing list using right click menu or double click
  5. User can be deleted but needs admin password which can be setup from menu
  6. User can be select pair types - Characters, Symbols or Smileys
  7. Game will be auto paused/resume on window lost/gain focus
  8. In each level player need to match 3 pairs
  9. Alphabets and Numbers will be shown on screen in different colors and orders
  10. As level increases score increases
  11. On wrong match user has to play level again
  12. On game start 3 second prepare screen will appear
  13. A game is for 1 min 20 sec time limit
  14. Settings menu is available from which
    1. User can select different colors and fonts to apply on screen
    2. User can change font size of game
    3. Users can see Help about game any time
  15. There are 3 tables on the screen
    1. First table shows Top-Scores of current user
    2. Second table shows Recent-Scores of current user
    3. Third table shows Top-Scores of all users from same machine
  16. User can Start/Stop game any time; On Stopping game scores will not count
